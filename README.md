# Musify

A full stack music streaming web application built using Django.

## Features
- User Login & Signup
- OTP Email Verification
- Play Songs
- Playlist Creation
- Trending Songs
- Music Player
- Responsive UI

## Technologies Used
- Python
- Django
- HTML
- CSS
- JavaScript
- Bootstrap

## Run Project

```bash
pip install -r requirements.txt
python manage.py runserver
```
